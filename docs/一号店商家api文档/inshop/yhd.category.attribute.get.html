API文档 类目API yhd.category.attribute.get

yhd.category.attribute.get 查询类别属性(不包含系列属性)

  • API接口描述
  • 系统级输入参数
  • 应用级输入参数
  • 返回结果
  • 返回示例
  • 错误码

API接口描述

根据类别编码和属性编码查询类别属性值。查询成功， 则返回成功记录数及相应的类别属性信息。查询失败，则返回查询失败记录数及相应的错误描述信息

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
categoryId Long 是 类别ID
attributeId Long 否 属性ID

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 1 查询成功记录数
errorCount Integer 是 0 更新失败记录数
categoryAttributeInfoList CategoryAttributeInfo[] 否 属性信息列表
errInfoList ErrDetailInfo[] 否 错误信息列表
