API文档 产品API ProdAttributeItemInfo

ProdAttributeItemInfo

简要描述

产品属性项信息,编辑类型为：下拉列表类型(单选,只能输入一个选项ID);多选框(多选,可以输入多个选项ID,|分隔),注意:有些属性是必填的

属性

名称 类型 是否必须 示例值 默认值 描述
attributeId Long 否 2882 属性ID(从获取类别属性接口中获取)
itemId Long[] 否 10708|10710 选项ID列表(竖杠分隔。如选项ID1|选项ID2,选项ID不能重复)(从获取类别属性接口中获取)
