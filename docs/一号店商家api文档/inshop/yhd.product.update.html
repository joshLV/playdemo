API文档 产品API yhd.product.update

yhd.product.update 更新一个产品(单品)

  • API接口描述

  • 应用级输入参数
  • 返回结果
  • 返回示例

API接口描述

根据输入产品的基本信息对产品进行更新操作，成功则返回成功记录数，失败则返回失败记录数和相应的错误描述信息。

      注：更新单个商品需提交商品审核

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
productId Long 否 1707612 无 一号店产品ID,与outerId二选一(productId优先)
outerId String 否 无 外部产品ID,与productId二选一
productCname String 否 无 产品名称(不超过100个字符)
productSubTitle String 否 无 产品名称副标题(不超过100个字符)
productNamePrefix String 否 无 产品名称前缀(不超过10个字符)
productDescription String 否 <img src="y.jpg" alt="y"/> 无 产品描述(文描)(不超过300kb)
prodAttributeInfoList ProdAttributeInfo[] 否 3281:电费1,3282: 产品属性列表(每个属性之间用逗号分隔,属性ID和属性值之间用冒号分隔,其中属性值可以不输入。如属性ID:属性值),属性id不能重复
prodAttributeItemInfoList ProdAttributeItemInfo[] 否 2882:,3280:11497|11499 属性项列表(每个属性之间用逗号分隔,属性ID和选项ID之间用冒号分隔,选项ID之间用竖线分隔,其中选项ID可以不输入。如属性ID:选项ID1|选项ID2),属性id不能重复,也不能同上面的属性ID相同,同一个属性ID不能有相同的选项ID

返回结果

名称 类型 是否必须 示例值 描述
updateCount Integer 是 1 更新成功记录数
errorCount Integer 是 0 更新失败记录数
errInfoList ErrDetailInfo[] 否 错误信息列表

返回示例

XML数据格式:

								<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
								<response>
									<errInfoList>
										<errDetailInfo>
											<errorCode>yhd.product.update.error</errorCode>
											<errorDes>更新产品失败(产品 API新增单品Test001 已存在)</errorDes>
										</errDetailInfo>
									</errInfoList>
									<errorCount>1</errorCount>
									<updateCount>0</updateCount>
								</response>

JSON数据格式:

{"response":{"errorCount":0,"updateCount":1}}

错误码

错误码 错误描述 解决方案
yhd.product.update.params_missing 参数:productId和outerId不能同时为空 两个参数必须指定一个
yhd.product.update.param_missing 需要更新的参数不能同时为空 输入需要更新的参数
yhd.product.update.param_list_missing 参数列表{0}格式或数据类型有误 检查参数列表格式和数据类型是否正确
yhd.product.update.error 更新产品失败({0}) 根据相应提示给出相应调整
yhd.product.update.system_error 系统运行异常 系统错误,请稍后再试
yhd.product.update.param_list_invalid 参数列表:{0}中的参数{1}有误 对应值不存在
yhd.product.update.param_invalid 参数:{0}有误 请输入正确的参数
yhd.product.update.param_length_over 参数:{0}超长(最大长度为{1}) 检查对应参数的长度是否正确
yhd.product.update.param_missed 必须属性:{0}不能更新为空 更新的时候不能将必须属性更新为空
