yhd.combine.products.search 查询套餐产品信息

API接口描述

用户可以通过canShow、canSale、curPage、pageRows来获取相应的套餐产品(即组合产品,由单品或系列子品组合在一起销售的产品)信息。查询成功，则返回查询成功记录数及相应的套餐产品信息。 查询失败，则返回失败记录数，及相应的错误信息描述
     

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
canShow Integer 否 是否可见(强制'上'下架),1是0否
canSale Integer 否 上下架状态0：下架，1：上架
curPage Integer 否 当前页数（默认1）
pageRows Integer 否 每页显示记录数（默认50、最大限制：100）

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 10 查询成功记录数
errorCount Integer 是 1 查询失败记录数
comProductList ComProduct[] 否 套餐产品列表
errInfoList ErrDetailInfo[] 否 错误信息列表
