API文档 订单API

订单API

提供了订单及订单相关信息的查询功能，更新订单卖家备注的功能

API列表

yhd.orders.get 获取订单列表（时间和订单状态）

yhd.order.detail.get 获取单个订单详情（含基本订单信息、产品信息、收货信息、优惠信息等）

yhd.orders.detail.get 批量获取订单详情

yhd.invoices.get 查询发票信息

yhd.order.merchant.remark.update 更新单个订单的卖家备注信息

数据结构列表

Order 订单

OrderDetail 订单详细信息

OrderItem 订单明细

OrderInfo 订单信息

Invoice 发票

ErrDetailInfo 错误信息
