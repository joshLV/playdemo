yhd.serial.products.search 查询系列产品信息

API接口描述

查询商家系列产品(系列产品是一系列由尺寸大小和颜色组成的产品的统称,即虚品)信息。查询成功，则返回查询成功记录数及相应的系列产品信息， 查询失败，则返回查询失败记录数，及相应的错误信息描述
     

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
curPage Integer 否 当前页数（默认1）
pageRows Integer 否 每页显示记录数（默认50、最大限制：100）

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 10 查询成功记录数
errorCount Integer 是 1 查询失败记录数
serialProductList SerialProduct[] 否 系列产品列表
errInfoList ErrDetailInfo[] 否 错误信息列表
