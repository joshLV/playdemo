API文档 类目API yhd.category.merchant.products.get

yhd.category.merchant.products.get 查询商家产品类别列表（店铺类别）

  • API接口描述
  • 系统级输入参数
  • 应用级输入参数
  • 返回结果
  • 返回示例
  • 错误码

API接口描述

根据商家id和父类别id 查询商家产品类别列表（店铺类别），只返回下一级。查询成功， 则返回更新成功记录数及相应的商家店铺里产品类别列表信息。查询失败，则返回查询失败记录数及相应的错误描述信息

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
categoryParentId Long 是 父类别ID（0：根节点）

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 1 查询成功记录数
errorCount Integer 是 0 更新失败记录数
merchantCategoryInfoList MerchantCategoryInfo[] 否 商家品牌信息列表
errInfoList ErrDetailInfo[] 否 错误信息列表
