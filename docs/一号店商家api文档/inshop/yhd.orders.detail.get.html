API文档 订单API yhd.orders.detail.get

yhd.orders.detail.get 批量获取订单详情

  • API接口描述
  • 系统级输入参数
  • 应用级输入参数
  • 返回结果
  • 返回示例
  • 错误码

API接口描述

通过指定订单编码来批量获取订单详细信息，查询成功，则返回查询成功记录数及相应的订单详细信息。 查询失败，则返回查询失败记录数及相应的错误信息描述

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
orderCodeList String[] 是 订单编码列表（逗号分隔）,最大长度为50

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 否 1 查询成功记录数
errorCount Integer 否 1 查询失败记录数
orderInfoList OrderInfo[] 否 订单详情列表
errInfoList ErrDetailInfo[] 否 错误信息列表
