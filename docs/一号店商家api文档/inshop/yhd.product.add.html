yhd.product.add 新增一个产品(单品)

API接口描述

增加一个产品(普通产品、图书)，成功则返回成功记录数，失败则返回失败记录数和相应错误描述信息。(只有图书产品才能输入图书特有属性)

      注：新增商品需提交商品审核

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
productType Integer 否 0 0 产品类型(0普通产品;1图书)
categoryId Long 是 957436 产品类别ID(从获取授权产品类别接口中获取叶子ID)
merchantCategoryId Long 是 12931 商家产品类别ID(从获取商家产品类别接口中获取叶子ID)
productCname String 是 产品名称(不超过100个字符)
productSubTitle String 否 产品名称副标题(不超过100个字符)
productNamePrefix String 否 产品名称前缀(不超过10个字符)
brandId Long 是 909986 品牌ID(从获取授权品牌接口中获取)
outerId String 是 out048 外部产品编码(不超过30个字符)
productMarketPrice Double 是 市场价
productSalePrice Double 是 销售价(不能大于市场价)
weight Double 是 重量(毛重KG)
virtualStockNum Long 是 库存(大于或等于0)
canSale Integer 否 0 1 是否可销(0否;1是)
productDescription String 否 <a href="#">y</a> 产品描述(不超过300kb)
electronicCerticate String 是 电子凭证(是或否)
prodAttributeInfoList ProdAttributeInfo[] 否 3281:电费1,3282:电费2 产品属性列表(每个属性之间用逗号分隔,属性ID和属性值之间用冒号分隔。如属性ID:属性值),属性id不能重复
prodAttributeItemInfoList ProdAttributeItemInfo[] 否 2882:10708,3280:11497|11499 属性项列表(每个属性之间用逗号分隔,属性ID和选项ID之间用冒号分隔,选项ID之间用竖线分隔。如属性ID:选项ID1|选项ID2),属性id不能重复,也不能同上面的属性ID相同,同一个属性ID不能有相同的选项ID
bookTitle String 否 标题(图书独有属性)
recommended String 否 出版社推荐语(图书独有属性)
authorIntroduction String 否 作者简介(图书独有属性)
catalog String 否 目录(图书独有属性)
digest String 否 书摘(图书独有属性)
contentIntroduction String 否 内容简介(图书独有属性)
mediaReport String 否 媒体报道(图书独有属性)

返回结果

名称 类型 是否必须 示例值 描述
updateCount Integer 是 1 更新成功记录数
errorCount Integer 是 0 更新失败记录数
errInfoList ErrDetailInfo[] 否 错误信息列表

返回示例

XML数据格式:

								<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
								<response>
									<errInfoList>
										<errDetailInfo>
											<errorCode>yhd.product.add.error</errorCode>
											<errorDes>新增产品失败(产品 API新增单品Test001 已存在)</errorDes>
										</errDetailInfo>
									</errInfoList>
									<errorCount>1</errorCount>
									<updateCount>0</updateCount>
								</response>

JSON数据格式:

{"response":{"errorCount":0,"updateCount":1}}

错误码

错误码 错误描述 解决方案
yhd.product.add.params_missing 输入的参数和产品类型不相符 检查是否输入了图书产品信息
yhd.product.add.param_missing 参数{0}类型错误或为空 输入指定参数
yhd.product.add.param_invalid 参数{0}有误 该参数不存在,检查对应参数是否正确
yhd.product.add.param_list_missing 参数列表{0}格式或数据类型有误 检查对应参数列表格式、数据类型是否正确
yhd.product.add.error 新增产品失败({0}) 根据对应原因调整相应参数
yhd.product.add.system_error 系统运行异常 系统错误,请稍后再试
yhd.product.add.param_list_invalid 参数列表:{0}中的参数{1}有误 对应值不存在(或检查数字类型的是否填写错误,编辑类型是否正确)
yhd.product.add.param_outerid_is_existed 外部产品编码已经存在 更换外部产品编码
yhd.product.add.others 其它:{0} 根据相应提示做出相应调整
yhd.product.add.param_length_over 参数:{0}超长(最大长度为{1}) 检查指定参数的长度
yhd.product.add.param_missed 必须属性:{0}没有输入 请在属性列表或属性项列表中输入指定的属性
