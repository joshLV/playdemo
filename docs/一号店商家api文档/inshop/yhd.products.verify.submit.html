yhd.products.verify.submit 批量提交商品审核

  • API接口描述

  • 返回结果

API接口描述

根据产品ID或外部产品编码提交产品审核，成功则返回成功记录数，失败则返回失败记录数及相应的错误描述信息。 (只有单品、套餐子品和虚品才能提交商品审核,套餐产品和系列子品不可以)

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
productIdList Long[] 否 1814540,1794134 一号店产品Id列表(逗号分隔),与outerIdList二选一(productCodeList优先),相同的产品id做一个产品id处理,最多100条
outerIdList String[] 否 外部产品编码列表(逗号分隔),与productIdList二选一,相同的外部产品编码做1个处理,最多100条

返回结果

名称 类型 是否必须 示例值 描述
updateCount Integer 是 1 更新成功记录数
errorCount Integer 是 0 更新失败记录数
errInfoList ErrDetailInfo[] 否 错误信息列表

返回示例

XML数据格式:

								<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
								<response>
									<errInfoList>
										<errDetailInfo>
											<errorCode>yhd.products.verify.submit.others</errorCode>
											<errorDes>其它:API新增单品0723提交审核002 缺少默认图片</errorDes>
											<pkInfo>API新增单品0723提交审核002</pkInfo>
										</errDetailInfo>
									</errInfoList>
									<errorCount>1</errorCount>
									<updateCount>0</updateCount>
								</response>

JSON数据格式:

{"response":{"errorCount":0,"updateCount":1}}

错误码

错误码 错误描述 解决方案
yhd.products.verify.submit.params_missing 参数:productIdList和outerIdList不能同时为空 两个参数必须指定一个
yhd.products.verify.submit.param_list_missing 参数列表{0}有误或超长(列表最大长度为{1}) 检查参数列表格式、数据类型和长度是否正确
yhd.products.verify.submit.outerid_repeat 系统中有重复的外部产品编码 修改对应产品的外部产品编码
yhd.products.verify.submit.param_invalid 参数{0}有误 检查相应参数或产品类型是否正确
yhd.products.verify.submit.system_error 系统运行异常 系统运行异常，请稍后再试
yhd.products.verify.submit.others 其它:{0} 根据相应提示调整参数
yhd.products.verify.submit.error 批量提交商品审核失败({0}) 根据相应提示做出修改
