yhd.group.buy.vouchers.get 查询消费券信息

API接口描述

团购平台调用此服务，输入相应的参数查询订单相关的消费券信息。查询成功，则返回 成功记录数，及相应的产品信息列表。查询失败，则返回失败记录数，及相应的错误信息描述

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
orderCode String 是 1号店订单号码
partnerOrderCode String 是 合作方订单号码

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 1 查询成功记录数
errorCount Integer 是 0 更新失败记录数
voucherInfoList VoucherInfo[] 否 产品信息列表
errInfoList ErrDetailInfo[] 否 错误信息列表

返回示例

XML数据格式:

								<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<response>
	<errorCount>0</errorCount>
	<totalCount>1</totalCount>
</response>

JSON数据格式:

{"response":{"errorCount":0,"totalCount":1}}

错误码

错误码 错误描述 解决方案
yhd.group.buy.vouchers.get.param_invalid 参数yhdOrderCode或yhdOrderCode为空 yhdOrderCode，yhdOrderCode必填
yhd.group.buy.vouchers.get.vouchers_not_found 消费券列表信息不存在 检查yhdOrderCode，yhdOrderCode是否正确
yhd.group.buy.vouchers.get.error 获取消费券列表信息失败 系统运行异常，请稍后再试
