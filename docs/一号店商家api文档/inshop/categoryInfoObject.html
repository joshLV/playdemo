CategoryInfo

简要描述

产品类别信息

属性

名称 类型 是否隐私 示例值 描述
categoryId Long 否 产品类别id
categoryName String 否 类别名称
categoryIsLeaf Integer 否 是否叶子节点 0:否 1:是
categoryCode String 否 类别编码
categoryParentId Long 否 父类别id
categoryDescription String 否 分类描述
categoryKeyword String 否 分类关键字
listOrder Integer 否 排序顺序
backOperatorId String 否 对销售价格的特殊管理人id，用逗号分割
categorySearchName String 否 搜索用的冗余字段:0-c1id-c2id-c3id-c4id:c1name-c2name-c3name-c4name
forumId Long 否 论坛id
isVisible Integer 否 是否可见 0 不可见 1可见
