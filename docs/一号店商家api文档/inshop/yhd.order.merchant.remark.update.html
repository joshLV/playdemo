yhd.order.merchant.remark.update 更新单个订单的卖家备注信息

  • API接口描述

  • 应用级输入参数
  • 返回结果

API接口描述

根据订单编码，订单卖家备注，更新单个订单的卖家备注信息，更新成功，返回更新成功记录数，更新失败，则返回更新失败记录数 及相应的错误信息描述

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
orderCode String 否 订单编码
remark String 否 订单卖家备注,最大长度为500

返回结果

名称 类型 是否必须 示例值 描述
updateCount Integer 否 10 更新成功记录数
errorCount Integer 否 1 更新失败记录数
errInfoList ErrDetailInfo[] 否 错误信息列表

返回示例

XML数据格式:

<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<response>
	<errorCount>0</errorCount>
	<updateCount>1</updateCount>
</response>

JSON数据格式:

{"response":{"errorCount":0,"updateCount":1}}

错误码

错误码 错误描述 解决方案
yhd.order.merchant.remark.update.param_missing 参数orderCode和remark没有输入 输入参数
yhd.order.merchant.remark.update.error 更新订单卖家备注失败 根据具体错误信息修改
yhd.order.merchant.remark.update.param_remark_over 备注信息超长(最大长度为500) 修改备注信息
