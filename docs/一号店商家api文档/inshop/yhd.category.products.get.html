API文档 类目API yhd.category.products.get

yhd.category.products.get 查询商家被授权产品类别列表

  • API接口描述
  • 系统级输入参数
  • 应用级输入参数
  • 返回结果
  • 返回示例
  • 错误码

API接口描述

根据商家id及父类别id查询商家被授权产品类别列表，只返回下一级。查询成功， 则返回更新成功记录数及相应的产品类别列表信息。查询失败，则返回查询失败记录数及相应的错误描述信息

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
categoryParentId Long 是 父类别ID（0：根节点）

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 1 查询成功记录数
errorCount Integer 是 0 更新失败记录数
categoryInfoList CategoryInfo[] 否 产品信息列表
errInfoList ErrDetailInfo[] 否 错误信息列表

返回示例

XML数据格式:

								<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<response>
	<errorCount>0</errorCount>
	<categoryInfoList>
		<categoryInfo>
			<categoryCode>U   </categoryCode>
			<categoryId>5135</categoryId>
			<categoryIsLeaf>0</categoryIsLeaf>
			<categoryParentId>0</categoryParentId>
			<categorySearchName>0-5135:食品饮料11</categorySearchName>
			<forumId>3493</forumId>
			<isVisible>1</isVisible>
			<listOrder>8</listOrder>
		</categoryInfo>
	</categoryInfoList>
	<totalCount>1</totalCount>
</response>

JSON数据格式:

{"response":{"categoryInfoList":{"categoryInfo":[{"categoryCode":"U   ","categoryId":5135,"categoryIsLeaf":0,"categoryParentId":0,"categorySearchName":"0-5135:食品饮料11","forumId":3493,"isVisible":1,"listOrder":8}]},"errorCount":0,"totalCount":1}}

错误码

错误码 错误描述 解决方案
yhd.category.products.get.param_invalid 参数categoryParentId类型错误或为空 输入categoryParentId，categoryParentId应指定为Long类型
yhd.category.products.get.category _not_found 产品类别列表信息不存在 检查categoryParentId是否正确
yhd.category.products.get.error 获取产品类别列表信息失败 系统运行异常，请稍后再试
