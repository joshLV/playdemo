API文档 产品API yhd.product.attribute.get

yhd.product.attribute.get 获取商品属性

  • API接口描述
  • 系统级输入参数
  • 应用级输入参数
  • 返回结果
  • 返回示例
  • 错误码

API接口描述

根据产品编码列表或外部编码列表查询商品的属性。查询成功， 则返回成功记录数及相应的商品属性列表信息。查询失败，则返回查询失败记录数及相应的错误描述信息

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
productIdList Long[] 否 一号店产品ID列表（逗号分隔，优先使用） 与outerIdList二选一,最大长度为100
outerIdList String[] 否 外部产品ID列表（逗号分隔） 与productIdList二选一,最大长度为100

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 1 查询成功记录数
errorCount Integer 是 0 更新失败记录数
productAttrInfoList ProductAttrInfo[] 否 产品属性信息列表
errInfoList ErrDetailInfo[] 否 错误信息列表

错误码

错误码 错误描述 解决方案
yhd.product.attribute.get.param_invaild 参数{0}类型错误或为空 检查传入的参数是否正确
yhd.product.attribute.get.param_length_over productCodeList或outerIdList列表超长(最大长度为100) 修改参数列表
yhd.product.attribute.get.prod_not_found 通过指定的参数无法查询到相应的产品属性信息 检查传入的参数是否正确
