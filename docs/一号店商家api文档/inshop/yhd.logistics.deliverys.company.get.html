API文档 物流API yhd.logistics.deliverys.company.get

yhd.logistics.deliverys.company.get 获取1号店合作物流公司信息（1号店自配送、3PL等）

  • API接口描述

  • 应用级输入参数

API接口描述

获取目前可用的1号店合作物流公司信息（包括快递公司编号，快递公司名称，快递公司查询地址，状态）

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
无

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 10 物流总记录数
errorCount Integer 是 0 查询失败记录数
logisticsInfoList LogisticsInfo[] 否 物流信息列表
errInfoList ErrDetailInfo[] 否 错误信息列表
