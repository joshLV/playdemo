API文档 参数类型列表

参数类型说明

 

Boolean

 

true or false,例如：true

 

Date

 

标准时间格式：yyyy-MM-dd HH:mm:ss，例如：2010-11-11 11:11:11

 

String

 

字符串，例如："城市"

 

String[]

 

以逗号分割的字符串列表，例如："城市,北京,省会"

 

Long

 

数字串，例如：1234,34343,2313

 

Long[]

 

数字串，例如：1234,34343,2313

 

Integer

 

例如：12345

 

Double

 

双精度类型，例如：12.50

 

Double[]

 

以逗号分割的双精度数字串列表，例如："12.50,14.50,37.43"

 

byte[]

 

字节数组
