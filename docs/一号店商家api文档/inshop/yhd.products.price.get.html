API文档 产品API yhd.products.price.get

yhd.products.price.get 批量获取产品价格信息

  • API接口描述

API接口描述

用户可以通过productIdList、outerIdList来批量获取相应的产品价格信息。查询成功，则返回查询成功记录数及相应的产品价格信息。 查询失败，则返回失败记录数，及相应的错误信息描述
备注：通过系列产品id是无法获取价格信息的，只有普通产品、套餐产品、套餐产品的子品、系列产品的子品可以获取价格信息      

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
productIdList Long[] 否 一号店产品ID列表（逗号分隔）,与outerIdList二选一,最大长度为100
outerIdList String[] 否 外部产品ID列表（逗号分隔）,与productIdList二选一,最大长度为100

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 10 查询成功记录数
errorCount Integer 是 1 查询失败记录数
pmPriceList PmPrice[] 否 产品价格列表
errInfoList ErrDetailInfo[] 否 错误信息列表

错误码

错误码 错误描述 解决方案
yhd.products.price.get.param_missing 参数：productIdList或outerIdList一个都没有传入 productIdList或outerIdList必须指定一个
yhd.products.price.get.param_invalid 参数：productIdList有误 productId应该指定Long型
yhd.products.price.get.param.error 通过指定的参数无法查询到相应的产品价格信息 检查传入的参数是否正确
yhd.products.price.get.error 批量获取产品价格信息接口错误 系统运行异常，请稍后再试
yhd.products.price.get.param_length_over productIdList或outerIdList列表超长(最大长度为100) 修改参数列表
