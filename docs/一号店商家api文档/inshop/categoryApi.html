API文档 类目API

类目API

提供查询商家产品类别，品牌类型，类别属性的操作

API列表

yhd.category.products.get 查询商家被授权产品类别列表

yhd.category.merchant.products.get 查询商家产品类别列表（店铺类别）

yhd.category.brands.get 查询商家被授权品牌列表

yhd.category.attribute.get 查询类别属性(不包含系列属性)

数据结构列表

CategoryInfo 产品类别信息

MerchantCategoryInfo 商家产品类别信息

BrandInfo 品牌信息

CategoryAttributeInfo 类别属性信息

AttributeItemInfo 选项属性信息

ErrDetailInfo 错误信息
