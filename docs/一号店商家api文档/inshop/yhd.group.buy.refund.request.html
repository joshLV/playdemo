API接口描述

用户在1号店发起退款申请，团购平台调用此服务，通知合作方，给用户退款

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
yhdOrderCode String 是 1号店订单号码
partnerOrderCode String 是 合作方订单号码
refundAmount Integer 是 退款金额
refundRequestTime String 是 2011-12-12 20:10:10 退款请求时间
格式：yyyy-MM-dd HH:mm:ss

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 10 申请成功记录数
errorCount Integer 是 1 申请失败记录数
RefundInfo RefundInfo 否 受理退款单信息
errInfoList ErrDetailInfo[] 否 错误信息列表
