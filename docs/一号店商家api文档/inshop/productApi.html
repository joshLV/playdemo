API文档 产品API

产品API

产品相关功能（包括查询，更新产品的价格信息，上下架信息，库存信息，套餐产品信息等）

API列表

yhd.product.add 新增一个产品（单品，需要提交商品审核）

yhd.general.products.search 查询普通产品的信息

yhd.serial.products.search 查询系列产品信息

yhd.serial.product.get 获取单个系列产品的子品信息

yhd.combine.products.search 查询套餐产品信息

yhd.combine.product.get 查询单个套餐子品信息

yhd.products.stock.get 批量获取产品库存信息

yhd.products.price.get 批量获取产品价格信息

yhd.product.stock.update 更新单个产品库存信息

yhd.product.price.update 更新单个产品价格信息（需要提交商品审核）

yhd.product.shelvesup.update 上架单个产品

yhd.product.shelvesdown.update 下架单个产品

yhd.product.outerid.update 更新单个产品外部编码id（需要提交商品审核）

yhd.product.attribute.get 获取商品属性

yhd.product.description.get 获取商品文描

yhd.products.verify.submit 批量提交商品审核

yhd.product.img.upload 添加商品图片（需要提交商品审核）

yhd.product.update 更新一个产品（单品，需要提交商品审核）

数据结构列表

ProdAttributeInfo 产品属性信息

ProdAttributeItemInfo 产品属性项信息

Product 商家产品

PmPrice 产品价格

PmStock 产品库存

ComProduct 套餐产品

ComChildProd 套餐子产品

SerialProduct 系列产品

SerialChildProd 系列子产品

ProductAttrInfo 产品属性信息

ProductDescInfo 产品文描信息

AttributeItemInfo 选项属性信息

ErrDetailInfo 错误信息
