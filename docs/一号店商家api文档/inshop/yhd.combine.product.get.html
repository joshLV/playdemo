API文档 产品API yhd.combine.product.get

yhd.combine.product.get 查询单个套餐产品的子品信息

  • API接口描述
  • 系统级输入参数
  • 应用级输入参数
  • 返回结果
  • 返回示例
  • 错误码

API接口描述

查询指定的套餐产品所有子品的信息。查询成功，则返回查询成功记录数及相应的套餐子产品信息， 查询失败，则返回查询失败记录数，及相应的错误信息描述
     

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
productId Long 否 一号店产品id,与outerId二选一
outerId String 否 外部产品ID 与productId二选一

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 1 查询成功记录数
errorCount Integer 是 0 查询失败记录数
comChildProdList ComChildProd[] 否 套餐子产品信息
errInfoList ErrDetailInfo[] 否 错误信息列表

返回示例

XML数据格式:

<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<response>
	<totalCount>1</totalCount>
	<errorCount>0</errorCount>
	<comChildProdList>
		<comChildProd>
			<canSale>1</canSale>
			<canShow>1</canShow>
			<categoryId>957344</categoryId>
			<ean13>6930511124206</ean13>
			<productCname>ACA烤箱ATO-M16A</productCname>
			<productCode>0000011858</productCode>
			<productId>1185</productId>
			<stockStatus>0</stockStatus>
			<isMainProduct>1</isMainProduct>
			<productNum>2</productNum>
		</comChildProd>
	</comChildProdList>
</response>

JSON数据格式:

{"response":{"comChildProdList":{"comChildProd":[{"canSale":1,"canShow":1,"categoryId":957344,"ean13":"6930511124206","isMainProduct":1,"productCname":"ACA烤箱ATO-M16A","productCode":"0000011858","productId":1185,"productNum":2,"stockStatus":0}]},"errorCount":0,"totalCount":1}}

错误码

错误码 错误描述 解决方案
yhd.combine.product.get.param_missing 参数productId和outerId不能同时为空 productId或outerId必须指定一个
yhd.combine.product.get.error 获取单个套餐子品信息失败 系统运行异常，请稍后再试
yhd.combine.product.get.prod_not_found 通过指定的参数无法查询到相应的产品信息 更换参数
