API文档 产品API yhd.product.img.upload

yhd.product.img.upload 添加商品图片

  • API接口描述
  • 系统级输入参数
  • 应用级输入参数
  • 返回结果
  • 返回示例
  • 错误码

API接口描述

根据输入的参数给指定的产品上传产品图片，上传成功则返回更新成功记录数； 上传失败则返回失败记录数和相应的错误描述信息。

      注：上传图片需提交商品审核，并且个数不能小于1张，不能超过5张，每张图片的大小不能超过1M。

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
productId Long 否 无 一号店产品ID,与outerId二选一(productId优先)
outerId String 否 无 外部产品ID,与productId二选一
mainImageName String 否 无 主图名称

返回结果

名称 类型 是否必须 示例值 描述
updateCount Integer 是 1 更新成功记录数
errorCount Integer 是 0 更新失败记录数
errInfoList ErrDetailInfo[] 否 错误信息列表

返回示例

XML数据格式:

								<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<response>
	<errorCount>0</errorCount>
	<updateCount>1</updateCount>
</response>

JSON数据格式:

{"response":{"errorCount":0,"updateCount":1}}

错误码

错误码 错误描述 解决方案
yhd.product.img.upload.param_missing 参数:productId和outerId不能同时为空 两个参数必须指定一个
yhd.product.img.upload.not_found 指定的产品信息不存在 输入指定参数
yhd.system.img.size_over 上传的图片大小超过1M 图片不能超过1M
yhd.system.img.num_null 上传的图片数量不能为0 修改上传图片数量
yhd.system.img.num_over 上传图片数量超过5个 修改上传图片数量
yhd.system.img.upload_error 图片上传失败 系统运行错误，请稍后上传
