yhd.general.products.search 查询普通产品的信息

API接口描述

用户可以通过canShow、canSale、productCname来获取相应的普通产品(没有尺寸大小和颜色之分的产品)信息。查询成功，则返回查询成功记录数及相应的产品信息， 查询失败，则返回查询失败记录数，及相应的错误信息描述
     

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
canShow Integer 否 是否可见(强制'上'下架),1是0否
canSale Integer 否 上下架状态0：下架，1：上架
curPage Integer 否 当前页数（默认1）
pageRows Integer 否 每页显示记录数（默认50、最大限制：100）
productCname String 否 商家产品中文名称(不支持模糊查询)

返回结果

名称 类型 是否必须 示例值 描述
totalCount Integer 是 10 查询成功记录数
errorCount Integer 是 1 查询失败记录数
productList Product[] 否 产品信息
errInfoList ErrDetailInfo[] 否 错误信息列表

错误码

错误码 错误描述 解决方案
yhd.general.products.search.param_invalid 参数：canShow或者 canSale有误 canShow、canSale、curPage、pageRows应该指定Integer型
yhd.general.products.search.error 查询普通商品信息失败 系统运行异常，请稍后再试
yhd.general.products.search.prod_not_found 通过指定的参数无法查询到相应的产品信息 检查是否有普通产品,检查传入的参数是否正确
