团购API

提供团购信息，消费券信息的创建，查询等操作

合作方API列表

yhd.group.buy.product.add 团购报名信息上传

yhd.group.buy.products.get 团购报名查询

yhd.group.buy.order.verify 团购订单信息确认

yhd.group.buy.refund.confirm 消费券退款确认

一号店团购平台API列表

yhd.group.buy.order.inform 团购订单信息通知

yhd.group.buy.vouchers.get 查询消费券信息

yhd.group.buy.voucher.resend 消费券短信重新发送

yhd.group.buy.refund.request 消费券退款申请

数据结构列表

groupProdInfo 团购报名信息

VoucherInfo 消费券信息

RefundInfo 受理退款单信息

ErrDetailInfo 错误信息
